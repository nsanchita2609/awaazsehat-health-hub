# AwaazSehat — AI-Powered Multilingual TB Screening Web App

AwaazSehat is a Progressive Web App (PWA) that enables TB risk screening 
via voice input in Hindi, Tamil, and Telugu — built for 500M+ low-literacy 
rural users with zero English proficiency.

## Features
- Voice-based TB symptom screening in Hindi, Tamil, and Telugu
- AI-powered risk assessment using Claude API
- Interactive clinic finder with map (Leaflet.js)
- Appointment booking and admin dashboard
- Voice readback of results in regional languages
- Fully offline-capable as a PWA

# Tech Stack
React.js, TailwindCSS, Node.js, Supabase, PostgreSQL, 
Claude API, Leaflet.js, Web Speech API, Vite, Vercel
